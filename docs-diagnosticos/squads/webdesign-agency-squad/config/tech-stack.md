# Tech Stack — webdesign-agency-squad

## Design & Prototipagem
- **Figma** — design system, wireframes, prototipagem, handoff
- **Adobe Creative Suite** — Photoshop, Illustrator, After Effects
- **Spline** — elementos 3D no browser

## Frontend
- **Next.js 14+** — framework React com SSR/SSG
- **React 18+** — biblioteca de UI
- **TypeScript** — tipagem estática
- **Tailwind CSS** — utility-first CSS
- **CSS Modules** — estilos locais por componente

## Animação & Motion
- **GSAP** + ScrollTrigger — animações de alta performance
- **Framer Motion** — animações React declarativas
- **Lottie** — animações After Effects na web

## 3D & Imersivo
- **Three.js** — 3D interativo com WebGL
- **@splinetool/runtime** — integração Spline no código
- **WebGL Shaders** — efeitos visuais avançados

## Backend & CMS
- **Node.js** — runtime backend
- **Python** — scripts e automações
- **GraphQL** + **REST APIs** — comunicação cliente-servidor
- **Sanity** / **Strapi** / **Contentful** — CMS headless

## Banco de Dados
- **PostgreSQL** — dados relacionais
- **MongoDB** — dados não-relacionais
- **Redis** — cache e sessões

## DevOps & Hosting
- **Vercel** — deploy frontend (preferido)
- **AWS** / **Netlify** / **Cloudflare Pages** — alternativas
- **GitHub Actions** — CI/CD
- **Docker** — containerização

## Performance & Monitoramento
- **Lighthouse** — auditoria de performance
- **WebPageTest** — testes de velocidade
- **Cloudflare CDN** — distribuição de assets
- **Sentry** — monitoramento de erros

## Analytics & CRO
- **Google Analytics 4** — análise de comportamento
- **Hotjar** — heatmaps e session recordings
- **Google Search Console** — SEO e performance de busca
- **Optimizely** / **VWO** — A/B testing

## Acessibilidade
- **axe DevTools** — testes automáticos de acessibilidade
- **NVDA** / **VoiceOver** — leitores de tela para teste
- **Colour Contrast Analyser** — verificação de contraste

## IA & Automação
- **Claude API** (Anthropic) — AI features e chatbots
- **OpenAI API** — alternativa para AI features
- **Pinecone** — vector database para AI search
- **Langchain** — orquestração de fluxos de IA
