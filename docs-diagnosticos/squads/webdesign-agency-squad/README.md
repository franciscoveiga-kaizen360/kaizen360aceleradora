## Filosofia

> Sites artesanalmente criados com personalidade única, que conectam emocionalmente, convertem e se destacam.

- **Design-first**: cada decisão técnica serve à experiência
- **Performance obrigatória**: abaixo de 2s em qualquer dispositivo
- **Acessível por padrão**: WCAG 2.2 AA/AAA desde o início
- **AI-ready**: GEO, personalização e integrações inteligentes
- **Ágil**: sprints, entregas incrementais, melhoria contínua
