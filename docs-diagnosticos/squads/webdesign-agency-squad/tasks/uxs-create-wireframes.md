# *create-wireframes

Wireframes estratégicos que comunicam estrutura e comportamento antes do visual.

## Regra dos Wireframes

Wireframe aprovado = estrutura congelada.
Após aprovação, mudanças estruturais são change requests.
Isso protege o cronograma e o budget.

## Fidelidade por Etapa

- **Lo-fi**: ideação rápida, blocos de conteúdo
- **Mid-fi**: estrutura definida, conteúdo real, anotações
- **Hi-fi**: protótipo — isso vai para o UX Strategist
