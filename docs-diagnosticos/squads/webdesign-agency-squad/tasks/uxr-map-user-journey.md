# *map-user-journey

Visualiza a experiência completa do usuário — antes, durante e depois.

## Dimensões do Mapa de Jornada

- **Etapas**: fases da jornada (descoberta → consideração → decisão → uso → fidelização)
- **Ações**: o que o usuário faz em cada etapa
- **Pensamentos**: o que passa pela cabeça
- **Emoções**: como se sente (escala positivo/negativo)
- **Touchpoints**: onde interage com a marca
- **Oportunidades**: onde podemos surpreender positivamente
