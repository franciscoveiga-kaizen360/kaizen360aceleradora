# *set-creative-vision

Estabelece a ambição criativa do projeto e os padrões de qualidade obrigatórios.

## O que define a visão criativa

- **Ambição**: qual o nível de ousadia criativa para este projeto?
- **Diferenciação**: o que vai fazer este site ser único e memorável?
- **Emoção**: que sentimento deve evocar no usuário?
- **Padrão**: qual é o mínimo aceitável de qualidade?

## Exemplos de visão criativa

- "Site premiado nível Awwwards — full immersive com 3D"
- "Conversão máxima — cada elemento serve à venda"
- "Branding forte — identidade que redefine o segmento"
