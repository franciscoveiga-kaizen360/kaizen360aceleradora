# *define-tone-of-voice

Voz consistente constrói reconhecimento. Ausência de voz é invisibilidade.

## Exemplo de Espectro de Tom

```
Formal  ●————————○  Casual
Sério   ●—————○————  Bem-humorado
Técnico ○————●————  Acessível
Frio    ○——●—————  Empático
```

A marca vive em algum ponto de cada espectro — documentar onde.
