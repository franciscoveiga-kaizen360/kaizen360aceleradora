# *conversion-optimization

CRO não é adivinhar. É testar, medir e iterar com base em evidências.

## Alavancas de CRO

1. **Clareza**: o usuário entende imediatamente o que é oferecido?
2. **Relevância**: a oferta é certa para quem chegou por aqui?
3. **Valor**: o benefício é claro e maior que o risco percebido?
4. **Urgência**: há motivo para agir agora?
5. **Fricção**: quão fácil é converter?
