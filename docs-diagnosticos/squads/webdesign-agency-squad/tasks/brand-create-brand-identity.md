# *create-brand-identity

Identidade visual não é apenas logo. É um sistema visual coerente.

## Entregáveis de Identidade Visual

1. **Logo** — principal, horizontal, ícone, negativo
2. **Cores** — primária, secundária, neutra, semântica
3. **Tipografia** — display (headlines), text (corpo), code (se aplicável)
4. **Iconografia** — estilo e família de ícones
5. **Fotografia/Ilustração** — diretrizes de estilo visual
6. **Motion** — como a identidade se move (para sites)
