# *optimize-conversion-funnel

Cada etapa deve empurrar o usuário para a próxima. Sem fricção, sem distração.

## Princípios Anti-fricção

- **Formulários**: apenas campos obrigatórios — cada campo a menos = mais conversão
- **CTAs**: um CTA principal por seção — múltiplos geram paralisia
- **Trust signals**: social proof, garantias e certificações perto dos CTAs
- **Velocidade**: cada segundo a mais no carregamento = -7% de conversão
