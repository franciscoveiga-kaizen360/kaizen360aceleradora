# *write-page-copy

Todo texto do site escrito com intenção e propósito.

## Hierarquia de Copy por Página

1. **H1 (Headline)**: promessa principal — o usuário fica ou sai
2. **Subtítulo**: expande e qualifica a headline
3. **Corpo**: evidências, benefícios, contexto
4. **CTA**: ação clara e específica

## Tom de Voz Consistente

Cada página deve soar como a mesma voz, mesmo com conteúdo diferente.
