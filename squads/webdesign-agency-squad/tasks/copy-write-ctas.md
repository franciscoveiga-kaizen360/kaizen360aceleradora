# *write-ctas

O CTA é a última frase antes da conversão. Cada palavra elimina ou cria hesitação.

## CTAs por Etapa do Funil

| Etapa | Exemplo de CTA | Tom |
|-------|---------------|-----|
| Descoberta | "Ver nosso trabalho" | Curioso, sem pressão |
| Consideração | "Calcular meu projeto" | Engajado, explorador |
| Decisão | "Começar agora" | Determinado, confiante |
| Urgência | "Últimas vagas este mês" | Urgente, mas verdadeiro |
