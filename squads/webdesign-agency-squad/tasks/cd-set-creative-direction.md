# *set-creative-direction

Define a linguagem visual e criativa que guia todos os especialistas do projeto.

## O que o Creative Brief deve responder

1. Que emoção o site deve despertar?
2. Qual é o nível de ousadia criativa?
3. Quais referências inspiram (sem copiar)?
4. O que é explicitamente proibido?
5. Como se diferencia dos concorrentes?
