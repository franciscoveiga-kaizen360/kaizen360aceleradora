# *create-headlines

A headline é o único copy que todos leem. Cada palavra conta.

## Fórmulas de Headline Comprovadas

- **Benefício direto**: "Site que vende enquanto você dorme"
- **Transformação**: "De visitante a cliente em 3 cliques"
- **Pergunta**: "Seu site converte menos de 3%?"
- **Número + promessa**: "97% dos clientes voltam — descubra por quê"
- **Contraste**: "Design premium. Resultado mensurável."
