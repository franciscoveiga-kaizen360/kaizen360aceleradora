# *design-user-flows

Mapeia o caminho do usuário do primeiro clique à conversão.

## Tipos de Fluxo

- **Fluxo de compra/conversão**: o mais crítico — zero fricção
- **Fluxo de onboarding**: primeira experiência define retenção
- **Fluxo de busca**: encontrar o que precisa rapidamente
- **Fluxo de suporte**: resolver problema sem frustração
