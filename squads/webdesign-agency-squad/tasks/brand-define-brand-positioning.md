# *define-brand-positioning

Posicionamento claro = escolhas de negócio mais fáceis. Tudo flui daí.

## Modelo de Positioning Statement

```
Para [público-alvo específico]
que [necessidade ou problema],
[nome da marca] é [categoria]
que [diferencial único].
Diferente de [alternativa principal],
[marca] [prova do diferencial].
```
