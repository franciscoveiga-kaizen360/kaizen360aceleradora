# *create-kinetic-typography

Tipografia que se move com propósito. Um diferencial que se tornou identidade.

## Técnicas de Tipografia Cinética

- **Split text**: letras ou palavras animadas individualmente (GSAP SplitText)
- **Text reveal**: texto revelado por mask ou clip-path
- **Scramble**: efeito de letras aleatórias antes de revelar (hacker effect)
- **Morphing**: transição entre palavras com forma compartilhada
- **Counting**: números animados para estatísticas e dados
